
# Input

Use `Input` component to create a styled `<input>` element.
It accepts all native HTML attributes for the element.

```.jsx
<Input
  id='email'
  name='email'
  defaultValue='hello@example.com'
/>
```

```.jsx
<Label htmlFor='email'>Email Address</Label>
<Input
  id='email'
  name='email'
  defaultValue='hello@example.com'
/>
```
