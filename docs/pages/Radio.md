# Radio

Use the `Radio` component to set the HTML `<input type='radio' />` element. Use it with `Label` .

```.jsx
<Label fontSize={1}>
  <Radio checked />
  {' '}
  Selected
</Label>
<Label fontSize={1}>
  <Radio disabled />
  {' '}
  Can't choose this one
</Label>
<Label fontSize={1}>
  <Radio />
  {' '}
  Not selected
</Label>
```
