
# Image

Use the `<Image />` component to render a primitive image that saves the aspect ratio.

```.jsx
<Image
  alt='Orlando'
  src='http://www.priceline.com/home/public/assets/images/destinations/orlando.jpg'
/>
```

```.jsx
<Image
  alt='Orlando'
  src='http://www.priceline.com/home/public/assets/images/destinations/orlando.jpg'
  width='300'
  height='200'
/>
```
