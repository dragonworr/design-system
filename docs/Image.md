
# Image

Use the `<Image />` component to render a primitive image that saves the aspect ratio.

```.jsx
<Image
  alt='Orlando'
  src='https://source.unsplash.com/bITjK6W2Alw/3200x1800'
/>
```

```.jsx
<Image
  alt='Orlando'
  src='https://source.unsplash.com/bITjK6W2Alw/3200x1800'
  width='300'
  height='200'
/>
```
